# Papers

📑  A repository for writing and collaborating on $\LaTeX{}$ papers (collaboration-wide). You can find a guide on how use $\LaTeX$ locally and how to collaborate on paper writing on GitHub in our [documentation](https://capibara3.github.io/docs/public/onboarding/17_local_latex.html).

If you are part of the @COSMOS team, see [capibara3/cosmos-papers](https://github.com/capibara3/cosmos-papers).

## Writing Papers on GitHub

> 🚨 Please, don't edit or push files into the main branch. Create a branch with you name/github username to make edits on your computer and then create a pull request from your branch to the main one. After a review (which you *have to* request to the latest person that edited that section and the principal author of the paper) you main merge the PR into the main branch. Don't delete your branch by that time, because you will reuse it for further edits. Thank you!
