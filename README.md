A repository for writing and collaborating on collaboration-wide (CRD+GAMMA) papers.

You can find a guide on how to use $LaTeX$ locally and how to collaborate on paper writing via GitHUb in our [documentation](https://capibara3.github.io/docs/public/onboarding_17_local_latex.html).

> [!DANGER]
> Please, don't edit or push files into the main branch. Create a branch with the pattern `<name or github username>/<paper or section>` on your computer and then create pull request from your branch to the main one. After a review (which you *have to * request to the latest person that edited that section and the principal author of the paper or section) you may merge the PR into the main branch. Don't delete your branch by that time, because you will reuse it for further edits. Thank you!

If you are part of the @GAMMA Science Team, see [capibara3/gamma-papers](https://github.com/capibara3/gamma-papers).
